# WaveHunter Roadmap

## v0.1.0 (Bootstrap) - [CURRENT]
- Custom WAV parsing and trailing payload checks.
- Metadata, channels, bitplane, and basic entropy extractors.
- Core CLI implementation.
- Basic magic byte/regex scanners.
- HTML, JSON, and Console reports.

## v0.2.0 (Signal Operations)
- Advanced strides & sample interlacing checks.
- Gray code and Delta stream decoding.
- Audio sample order and bit-stream reversals.

## v0.3.0 (Heuristics & Compression)
- Automated extraction of compressed blocks.
- Improved flag scanning regex databases.
- Deeper signal heuristic scoring.

## v0.5.0 (Phase Analysis)
- DFT/FFT analysis of phase shifts.
- Echo-hiding analysis capabilities.

## v0.8.0 (Automation & SDK)
- Plugin SDK for custom user extractors/scanners.
- Batch processing of directory lists.

## v1.0.0 (Release)
- Test suite with high coverage.
- Fully stabilized CLI and report dashboards.
- Detailed documentation site.
